Thank You For Installing SAP!
Aiding in protection!
What is SAP?
SAP is a Chrome extension whose sole purpose is to help the user protect themselves when using Steam.
Where can I go for help?
You can click this link or click the "Help" button on the top bar! You can also access the direct link from the settings page.
Will SAP 100% protect me?
The simple answer is: No.
SAP was designed to consolidate information where necessary to help you protect yourself. If you don't know how to read the information SAP will display to you, you can read the help page to better understand it!
I have a suggestion/bug report!
SAP is an open-source project. If you have the technical knowhow, you can add your suggestion or fix your problem yourself!
If you don't know-how, you can always make an issue report on the issues page and we can see what we can do to fix it.
